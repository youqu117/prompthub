# PromptHub Pro 使用指南 (Windows)

PromptHub Pro 是一个先进的提示词管理工具。由于浏览器安全限制，它不能像普通的 `.exe` 文件一样直接双击运行，但你可以通过以下简单步骤将其变成一个**桌面应用程序**。

## 第一步：如何在 Windows 上运行

建议使用 **VS Code** 方式（最稳定）：

1. **准备环境**：下载并安装 [VS Code](https://code.visualstudio.com/)。
2. **打开代码**：在 VS Code 中点击 `文件` -> `打开文件夹`，选择本项目目录。
3. **安装插件**：在左侧插件市场搜索并安装 **"Live Server"**。
4. **启动服务**：在编辑器底部状态栏点击 **"Go Live"**（或者右键点击 `index.html` 选择 `Open with Live Server`）。
5. **访问应用**：浏览器会自动弹出应用界面。

---

## 第二步：将其变为“桌面软件” (PWA)

为了获得和普通软件一样的独立窗口体验：

1. 在 Chrome 或 Edge 浏览器打开应用后，点击浏览器右上角的 **三个点 (⋮)**。
2. 选择 **“应用” (Apps)** -> **“将此站点作为应用安装” (Install site as app)**。
3. 现在你的桌面上会出现 **PromptHub** 的图标，它可以脱离浏览器作为独立窗口运行。

---

## 第三步：核心功能操作指南

### 1. 管理工作流
- **新建**：点击右上角的“+ 新建工作流”。
- **编辑**：点击列表卡片即可在右侧展开。
- **快照/版本**：在编辑过程中，点击“创建快照”可以保存当前版本。点击编辑器顶部的数字（1, 2...）可以随时跳回之前的记录。

### 2. 分类管理
- **新增**：点击左侧“分类导航”旁的 `+`。
- **重命名/删除**：鼠标悬停在分类上，使用出现的图标进行操作。*注意：删除分类后，其中的提示词会自动归类到“通用”。*

### 3. 数据备份 (强烈建议)
- 软件数据保存在浏览器本地（LocalStorage）。
- 点击左下角 **“系统偏好设置”**。
- 定期使用 **“导出备份”** 功能下载数据文件，防止系统清理导致数据丢失。

---

## 技术支持
- 本应用无需联网即可管理数据（100% 本地存储）。
- 适配了 Windows 黑暗模式，随系统主题自动切换。
